SleepJiraBot

Управляй задачами Jira прямо из Telegram. Мгновенное создание со скриншотами, подписки в реальном времени, спринты, дейли и отчёты по расписанию — без переключения контекста.

Открыть в Telegram
🔗

OAuth-авторизация Jira

Безопасное подключение к Jira Cloud через OAuth 2.0. Поддержка нескольких сайтов Atlassian, автоматический refresh токенов.

⚡

Мгновенное создание (/createfast)

Отправь текст, фото или файл с подписью — бот создаст задачу в Jira с вложением. Подтверждение, выбор Epic и автодозаполнение дефолтов.

🔔

Подписки и вебхуки

Уведомления о новых задачах, упоминаниях и смене статусов — в реальном времени через вебхуки Jira (HMAC-SHA256) и поллинг.

🏃

Спринты и дейли-стендап

Доски активных спринтов и сводка по задачам на сегодня. Подписка на ежедневный стендап с учётом твоей таймзоны.

📋

Отчёты по расписанию

Автоматическая отправка JQL и спринт-отчётов по cron в любой чат или канал.

✏️

Полный workflow задач

Пошаговый мастер с Epic, приоритетом и исполнителем, комментарии, смена статуса, назначение — всё из Telegram.

🛠️

Дефолты и шаблоны

Настрой проект, борд и тип задачи по умолчанию через /defaults. Сохраняй переиспользуемые шаблоны с tap-to-copy телом.

🌐

Два языка

Полная поддержка русского и английского интерфейса. Переключение одной командой.

Как начать

1

Открой бота

Найди @Sleep_Jira_Bot в Telegram и нажми /start

2

Подключи Jira

Команда /connect откроет OAuth-авторизацию в Jira Cloud

3

Работай

Создавай задачи из скриншотов, подписывайся на проекты, смотри спринты, настрой дейли и отчёты

Команды

/start, /menu Главное меню
/connect Подключить аккаунт Jira Cloud
/disconnect Отключить аккаунт Jira
/me Информация о подключённом аккаунте
/daily [username] Сводка задач на сегодня — по себе или коллеге
/sprint [PROJECT] [BOARD] Просмотр активного спринта
/issue KEY Детали задачи по ключу
/list JQL Поиск задач через JQL-запрос
/create Пошаговый мастер создания задачи
/createfast Мгновенное создание: текст или фото/файл с подписью → задача с вложением
/comment KEY текст Добавить комментарий к задаче
/transition KEY Сменить статус задачи
/assign KEY Назначить исполнителя
/watch Подписаться на обновления
/unwatch Отписаться от обновлений
/subscriptions Список активных подписок
/filters Сохранённые Jira-фильтры
/schedule cron Настроить отчёт по расписанию
/schedules, /unschedule Управление расписаниями
/defaults Проект, борд и тип задачи по умолчанию
/lang Переключить язык (ru/en)
/help Справка по всем командам

🔐 Безопасность

Токены Jira шифруются AES-256-GCM и хранятся в MongoDB. Вебхуки Jira проверяются по подписи HMAC-SHA256. Бот не хранит пароли и не имеет доступа к данным без вашего OAuth-разрешения.

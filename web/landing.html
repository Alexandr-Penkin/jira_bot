SleepJiraBot

Управляй задачами Jira прямо из Telegram. Подписки, спринты, отчёты — без переключения контекста.

Открыть в Telegram
🔗

OAuth-авторизация Jira

Безопасное подключение к Jira Cloud через OAuth 2.0. Поддержка нескольких сайтов Atlassian.

🔔

Подписки на обновления

Следи за задачами, проектами и фильтрами. Получай уведомления о новых задачах, упоминаниях и изменениях статуса.

🏃

Спринты и дэйли

Просматривай активные спринты и получай сводку по своим задачам на сегодня одной командой.

📋

Отчёты по расписанию

Настрой автоматическую отправку JQL-отчётов по cron-расписанию в любой чат.

✏️

Действия с задачами

Комментируй, меняй статус, назначай исполнителя — всё из Telegram, без открытия браузера.

🌐

Два языка

Полная поддержка русского и английского интерфейса. Переключение одной командой.

Как начать

1

Открой бота

Найди @SleepJiraBot в Telegram и нажми /start

2

Подключи Jira

Команда /connect откроет OAuth-авторизацию в Jira Cloud

3

Работай

Подпишись на проекты, смотри спринты, получай отчёты

Команды

/connect Подключить аккаунт Jira Cloud
/disconnect Отключить аккаунт Jira
/me Информация о подключённом аккаунте
/daily Сводка задач на сегодня
/sprint Просмотр активного спринта
/issue KEY Детали задачи по ключу
/list JQL Поиск задач через JQL-запрос
/comment KEY текст Добавить комментарий к задаче
/transition KEY Сменить статус задачи
/assign KEY Назначить исполнителя
/watch Подписаться на обновления
/unwatch Отписаться от обновлений
/subscriptions Список активных подписок
/filters Сохранённые Jira-фильтры
/schedule Настроить отчёт по расписанию
/schedules Список отчётов по расписанию
/lang Переключить язык (ru/en)

🔐 Безопасность

Токены Jira шифруются AES-256-GCM и хранятся в MongoDB. Бот не хранит пароли и не имеет доступа к данным без вашего OAuth-разрешения.
